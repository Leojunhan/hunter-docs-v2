## 1. 需求背景

### 1.1 业务背景

Hunter 核心业务路径：

```
发现目标 → 清洗目标 → 触达目标 → 转化目标
```

### 1.2 Agent + Skill 映射

| 业务路径 | Agent | Skill |
|:--------|:------|:------|
| 发现目标 | 情报 Agent | FB 评论爬取 |
| 清洗目标 | 情报 Agent | NLP 分类 + 规则评分 + 去重 |
| 触达目标 | 执行 Agent | FB 评论回复 |
| 转化目标 | 执行 Agent | 转化引导 |

## 3. 业务流程

### 3.1 Agent 执行流程

```
情报 Agent 发现 → 情报 Agent 清洗 → 执行 Agent 触达 → 执行 Agent 转化
```

### 3.2 目标状态流转

```
发现 → 待清洗 → 已过滤 / 待分配 → 已分配 → 已触达 → 待回复 → 已转化 → 已关闭
```

| 状态 | 触发 | 处理方 |
|:----|:----|:------|
| 发现 | Copaw 匹配关键词 | 自动 |
| 待清洗 | 进入清洗队列 | 自动 |
| 已过滤 | 清洗不通过 | 自动 |
| 待分配 | 清洗通过 | 自动 |
| 已分配 | 指定 Agent | 手动优先 |
| 已触达 | Agent 执行 | 自动 |
| 待回复 | 等待用户响应 | 自动 |
| 已转化 | 外部系统回传 | 自动 |
| 已关闭 | 手动关闭 | 手动 |

## 5. 功能规格（后台管理端）

### 5.1 Agent 管理

- 列表：状态、在线时间、任务数
- 详情：基本信息 + 绑定剧本 + 执行日志
- 操作：启用/禁用

### 5.2 Skill 管理

- 列表：类型、调用次数、截流参数
- 详情：配置参数 + 执行日志
- 操作：截流参数配置（P0）

### 5.3 目标管理

- 列表：状态、评分、分配 Agent
- 详情：来源信息 + 清洗记录 + 触达记录
- 操作：手动分配（P0）

### 5.4 执行日志

- 列表：时间、Agent、Skill、结果
- 详情：输入输出数据

### 5.5 API 密钥

- 列表：名称、创建时间、状态
- 操作：添加/编辑/删除
