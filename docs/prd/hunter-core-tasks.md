## 1. 需求背景

核心业务路径：

```
发现目标 → 清洗目标 → 触达目标 → 转化目标
```

Agent + Skill 映射：

| 业务路径 | Agent | Skill |
|:--------|:------|:------|
| 发现目标 | 情报 Agent | FB 评论爬取 |
| 清洗目标 | 情报 Agent | NLP 分类 + 规则评分 + 去重 |
| 触达目标 | 执行 Agent | FB 评论回复 |
| 转化目标 | 执行 Agent | 转化引导 |

## 3. 业务流程

目标状态流转：

```
发现 → 待清洗 → 已过滤 / 待分配 → 已分配 → 已触达 → 待回复 → 已转化 → 已关闭
```

| 状态 | 触发 | 处理方 |
|:----|:----|:------|
| 发现 | Copaw 匹配关键词 | 自动 |
| 待清洗 | 进入清洗队列 | 自动 |
| 已过滤 | 清洗不通过 | 自动 |
| 待分配 | 清洗通过 | 自动 |
| 已分配 | 指定 Agent | 手动优先 |
| 已触达 | Agent 执行 | 自动 |
| 待回复 | 等待用户响应 | 自动 |
| 已转化 | 外部系统回传 | 自动 |
| 已关闭 | 手动关闭 | 手动 |

### 5.2 Skill 管理

### 5.3 目标管理

- 列表 + 详情 + 手动分配
- 操作：手动分配（P0）

### 5.4 执行日志

- 列表 + 详情

### 5.5 API 密钥

- 列表 + 添加/编辑/删除
