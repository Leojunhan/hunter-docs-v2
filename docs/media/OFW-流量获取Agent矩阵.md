## 1. 流量获取框架

### 1.1 核心洞察

**OFW获取信息的渠道：**
- WhatsApp（跟家人视频、日常沟通）
- Telegram（频道广播、大型社群）
- Facebook（刷社群、刷视频、讨论问题）
- Google（搜索问题答案）
- OFW社群（同乡会、教会群）
- App Store/Google Play（下载APP）

**核心认知：**
- OFW要的是**"谁可信"**，不是**"哪里有贷款"**
- 信任建立需要**时间**和**人格化**
- 流量只是**入口**，**信任**才是**转化**

**WhatsApp号管控问题：**
- WhatsApp对私域营销管控严格，容易封号
- 需要多账号矩阵，分散风险
- Telegram作为备用私域渠道，**优先级提升**

## 3. 各渠道Agent详细设计

### 3.1 Facebook群组（核心渠道）

#### 渠道特征

| 维度 | 分析 |
|:-----|:-----|
| OFW活跃度 | 高，每天刷FB |
| 信任属性 | 社群讨论，信任度高 |
| 转化场景 | "哪家贷款靠谱"、"没有信用记录能借吗" |
| 风险 | 群主踢人、广告封号 |

#### Agent：社群运营Agent

**职责：**
1. 发现并加入OFW相关群组
2. 观察群内话题和痛点
3. 回答贷款相关问题
4. 分享真实用户故事
5. 引导至WhatsApp私域

**具体动作：**

```
Step 1：发现群组
- 搜索关键词：OFW、Overseas Worker、Philippines loan
- 找到同乡会群、教会群、菲律宾劳工群
- 申请加入，观察1-2周

Step 2：建立存在感
- 参与群内话题，不直接提贷款
- 分享OFW生活相关的内容
- 混个脸熟

Step 3：回答问题
- 有人问"哪家贷款靠谱" → 客观分享经验
- 有人问"没有信用记录能借吗" → 介绍替代数据风控
- 有人问"贷款需要什么材料" → 详细解答

Step 4：引导私域
- "如果你有更多问题，可以加我WhatsApp详细聊"
- "我们专门服务OFW，理解你的处境"
- 不直接推销，先建立信任
```

**风险控制：**
- 不发广告，会被踢
- 不刷屏，每天最多回答3-5个问题
- 不夸大，客观分享

**核心指标：**
- 入群数量：>20个OFW群
- 回答问题：>50次/月
- 引导私域：>20人/月

## 6. 核心原则

### 6.1 流量原则

1. **信任第一**：流量只是入口，信任才是转化
2. **真诚第二**：OFW不傻，能看出来套路
3. **长期第三**：口碑需要时间积累

### 6.2 Agent原则

1. **自动化**：能Agent做的，不让人做
2. **人工兜底**：复杂问题、投诉处理需要人
3. **数据驱动**：用数据优化策略

```
好服务 → 用户满意 → 口碑推荐 → 低成本获客 → 更多用户
    ↑                                              ↓
    └──────────────── 复借 + 裂变 ←────────────────┘
```

## 8. 人格化获客策略（进阶）

> 本章节整合了进阶获客策略，部分表述较为激进，仅供参考。执行时需注意平台规则和合规风险。

### 8.1 核心战略：刚需拦截 → 视觉冲击 → 快速转化

| 阶段 | 核心动作 | 目标 |
|:-----|:---------|:-----|
| 刚需拦截 | 在用户最渴求钱的时候出现 | 主动触达，而非被动等待 |
| 视觉冲击 | 用真实感击碎心理防线 | 建立信任，消除疑虑 |
| 快速转化 | 极短链路，瞬间闭环 | 剔除冗余，直接转化 |

### 8.2 双线并发模式

#### 线路A：精英高净值线（The Elite Track）

| 项目 | 内容 |
|:-----|:-----|
| 目标市场 | 香港 (HK) |
| 核心终点 | Askbing.com.hk / cococredit.com |
| 获客逻辑 | 专业背书 → 财务弹性 → 隐私闭环 |
| 拦截阵地 | LinkedIn职场圈、金融论坛、高净值FB群组 |
| 人格部署 | 【人格A：资深财务顾问】- 不谈"缺钱"，谈"资金周转率"和"财务杠杆优化" |
| 视觉风格 | 极简、商务、高冷色调（蓝/金色） |
| 转化链路 | 专业问答 → 私信方案 → 引导至官网 |

> ⚠️ **风险提示**：LinkedIn对营销内容管控较严，账号容易被封，建议使用真实职业身份背书

#### 线路B：下沉海量规模线（The Mass Track）

| 项目 | 内容 |
|:-----|:-----|
| 目标市场 | 印尼 (ID) |
| 核心终点 | App Store / Google Play（pinjamlng, rupiahgo等） |
| 获客逻辑 | 情绪共鸣 → 成功诱导 → 极速下载 |
| 拦截阵地 | FB贷款互助群、Telegram借贷频道、本地社区贴吧 |
| 人格部署 | 【人格B：成功获批用户】×10（矩阵号）- 晒到账截图、讲故事 |
| 视觉风格 | 高饱和度、快节奏、Before/After对比 |
| 转化链路 | 故事分享 → 评论区引导 → 直接点击下载链接 |

> ⚠️ **风险提示**：FB/TG对贷款类营销管控严格，避免直接硬广，采用"分享经验"方式

### 8.3 人格矩阵设计

### 8.6 转化漏斗分析

| 阶段 | 精英线(HK) | 规模线(ID) | 分析 |
|:-----|:----------:|:----------:|:-----|
| 拦截→响应 | 12% | 45% | ID市场对"钱"的响应是生理性的 |
| 响应→互动 | 65% | 30% | HK市场需要深层沟通建立信任 |
| 互动→点击 | 20% | 70% | ID市场只要信任，点击极其果断 |
| 点击→申请 | 15% | 25% | 落地页加载速度是ID市场的生死线 |

> ⚠️ **数据说明**：以上为理论推算，实际CTR可能<5%，需以实测数据为准
