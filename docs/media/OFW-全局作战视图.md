## 2. 获客策略结构

### 2.1 策略分层

| 层级 | 策略 | 目标 | 核心动作 |
|:-----|:-----|:-----|:---------|
| **战略层** | 双线并发 | 覆盖不同用户群 | 精英线+规模线 |
| **战术层** | 刚需拦截 | 在痛点最高处触达 | 关键词监控、评论区拦截 |
| **执行层** | 人格矩阵 | 建立信任 | 专家/用户/预警者 |
| **转化层** | 快速闭环 | 极短链路 | 拦截→引导→下载/申请 |

### 2.2 双线策略对比

| 维度 | 精英线 (HK) | 规模线 (ID) |
|:-----|:-----------:|:-----------:|
| 目标市场 | 香港 | 印尼 |
| 核心产品 | Askbing.com.hk | Loan App |
| 获客逻辑 | 专业背书→财务弹性→隐私 | 情绪共鸣→成功诱导→极速 |
| 拦截阵地 | LinkedIn、金融论坛 | FB群组、Telegram |
| 人格 | 资深财务顾问 | 成功获批用户×10 |
| 视觉风格 | 商务蓝/金色 | 活力绿/橙 |
| 转化链路 | 专业问答→私信→官网 | 故事→评论→下载 |

### 2.3 渠道策略

| 渠道 | 策略 | Agent | 优先级 | 风险 |
|:-----|:-----|:------|:------:|:-----|
| Facebook群组 | 信任背书、回答问题 | 社群运营Agent | P0 | 中 |
| Google/SEO | 搜索拦截、攻略内容 | SEO内容Agent | P0 | 低 |
| WhatsApp | 私域沉淀、定期触达 | 私域运营Agent | P0 | 高(封号) |
| Telegram | 频道广播、社群运营 | 电报运营Agent | P1 | 低 |
| 评论区 | 客观分享、引导私信 | 评论区获客Agent | P1 | 中 |
| App Store | 引导好评、差评回访 | 好评引导Agent | P1 | 低 |
| TikTok | 短视频、视觉冲击 | 短视频Agent | P2 | 中 |

### 2.4 获客漏斗

```
曝光 (50,000) → 互动 (5,000) → 咨询 (1,500) → 申请 (750) → 放款 (300)
   10%          30%           50%           40%
```

### 2.5 关键策略动作

| 阶段 | 动作 | 目的 |
|:-----|:-----|:-----|
| 拦截 | 监控"急用钱"、"被拒"等关键词 | 精准触达 |
| 切入 | 用"方案建议"替代"产品推广" | 降低防备 |
| 建立 | 专家/用户/预警者人格轮换 | 多维信任 |
| 引导 | 私信/链接/下载多路径 | 适应不同用户 |
| 转化 | 极简链路、快速闭环 | 减少流失 |
