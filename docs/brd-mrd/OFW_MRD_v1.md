### 0.2 产品定位

**一句话定位：** 为 OFW 提供基于 AI Agent 的信任建立型信贷服务

| 维度 | 内容 |
|------|------|
| **目标用户** | 菲律宾海外劳工（OFW），25-45 岁，有稳定跨境汇款记录 |
| **核心痛点** | 有收入但无信用记录，银行贷不了款 |
| **解决方案** | AI Agent 对话建立信任 + 替代数据风控 + 小额短期贷款 |
| **差异化** | 不是降低门槛，是重构信任建立方式（对话语义数据替代征信报告） |
| **商业模式** | 利息收入（24%/年）+ 复借循环 |

## 1. 市场分析

### 1.1 市场规模（TAM/SAM/SOM）

| 层级 | 定义 | 规模 | 计算逻辑 |
|------|------|------|---------|
| **TAM** | 总可服务市场 | $35B/年 | 菲律宾 OFW 年汇款总额 |
| **SAM** | 可获取市场 | $500M/年 | 有借贷需求 OFW × 平均借款$500 × 12 次/年 |
| **SOM** | 可获得市场 | $5M/年（Year 3） | SAM × 1% 市场份额 |

**计算逻辑：**

```
TAM = 200 万 OFW × $17,500/年（平均收入） = $35B

SAM = 200 万 × 60%（有借贷需求） × $500（平均借款） × 12 次/年 = $500M

SOM（Year 3） = $500M × 1% = $5M
```

### 3.2 用户场景

**场景 1：紧急医疗借款**

```
用户：Maria，32 岁，新加坡家政工人

行为：
1. 在 Facebook 同乡群看到推荐

```

**场景 3：被拒用户留存**

```
用户：Ana，28 岁，新加坡护工

行为：
1. Agent 温暖告知被拒原因

                          ↓

                          ↓

```
