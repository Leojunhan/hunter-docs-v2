目标市场： 菲律宾海外劳工（OFW）信贷服务

可服务市场规模$500M/年（估算）内部测算⭐⭐ 中

市场机会本质：OFW 有稳定跨境收入，但本地征信空白，传统银行无法服务。

0.2 产品定位

一句话定位： 为 OFW 提供基于 AI Agent 的信任建立型信贷服务

维度内容
目标用户菲律宾海外劳工（OFW），25-45 岁，有稳定跨境汇款记录
核心痛点有收入但无信用记录，银行贷不了款
解决方案AI Agent 对话建立信任 + 替代数据风控 + 小额短期贷款
差异化不是降低门槛，是重构信任建立方式（对话语义数据替代征信报告）
商业模式利息收入（24%/年）+ 复借循环

3.2 用户场景

场景 1：紧急医疗借款

用户：Maria，32 岁，新加坡家政工人
触发：母亲生病，需要$500 医疗费
行为：
1. 在 Facebook 同乡群看到推荐
2. 点击链接，Messenger 对话开始
3. Agent 了解需求，引导申请
4. 上传证件 + 活体检测
5. AI 预审通过，人工终审
6. 2 小时内放款到账户
7. 3 个月后还款，复借额度提升

场景 2：教育借款

用户：Jose，35 岁，中东建筑工人
触发：孩子学费到期，需要$300
行为：
1. Google 搜索"OFW loan"
2. 看到 SEO 内容，点击进入
3. 填写表单申请
4. AI 预审 + 人工审批
5. 放款，按时还款
6. 成为复借用户

场景 3：被拒用户留存

用户：Ana，28 岁，新加坡护工
触发：首次申请被拒
行为：
1. Agent 温暖告知被拒原因
2. 提供改善建议（如：稳定汇款 3 个月）
3. D+30 发送状态更新鼓励
4. D+90 邀请再次申请
5. 二次申请通过，成为忠诚用户
